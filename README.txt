Portfolio v3 - Usage instructions (Arabic comments are inside files):
1) Unzip the folder and open index.html in your browser (or upload to hosting).
2) Replace files/links:
   - your-photo.gif  -> your avatar (jpg/png/gif)
   - your-video.mp4  -> background video (optional)
   - your-music.mp3  -> music file (or use external mp3 link in index.html)
   - your-server-image.png -> fallback server image for local use
   - https://discord.gg/yourcode -> replace with your actual discord invite
   - Social links in index.html
3) To auto-load server icon, upload to hosting (Netlify, Vercel, GitHub Pages).
4) If autoplay blocked, user interaction may be required to start audio.
